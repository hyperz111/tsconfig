# @hyperz111/tsconfig

Shared `tsconfig` file for [`@hyperz111`](https://github.com/hyperz111/).

## Install

```text
npm install -D @hyperz111/tsconfig
```

## Usage

In `tsconfig.json`:

```jsonc
{
	// ...
	"extends": "@hyperz111/tsconfig",
	// ...
}
```
